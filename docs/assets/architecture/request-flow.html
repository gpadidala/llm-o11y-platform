End-to-End Request Flow

10-Step Gateway Middleware Pipeline — Every LLM request traverses this path
Client Application / Agent / MCP Client
POST /v1/chat/completions
Authorization: Bearer sk-llmo-xxxxx
▼
1
Virtual Key Auth
Validate sk-llmo-xxx key, check budget, verify permissions
src/gateway/virtual_keys.py
▼
2
Rate Limiter
Token bucket + sliding window (RPM, RPH, RPD, TPM, TPD)
src/gateway/rate_limiter.py
▼
3
Guardrails (Input)
PII detection (18 patterns), content safety, topic restriction
src/guardrails/engine.py
▼
4
Cache Check
SHA-256 exact match or trigram cosine similarity (0.85 threshold)
src/gateway/cache.py → HIT: return cached response
▼ MISS
5
Circuit Breaker
Check provider health — OPEN: try fallback target
src/gateway/circuit_breaker.py (CLOSED/OPEN/HALF_OPEN)
▼
6
Routing Engine
Select target via strategy: cost, latency, fallback, canary, loadbalance
src/gateway/routing.py
▼
7
Retry with Backoff
Exponential backoff + jitter on 429, 500-504 errors (max 3 retries)
src/gateway/retry.py
▼
8
Provider Adapter
Translate to native API: openai, anthropic, vertexai, boto3, cohere
src/providers/{openai,anthropic,vertex,bedrock,cohere}_provider.py
▼
9
Cache Store
Store response for future requests (TTL: 1 hour default)
src/gateway/cache.py
▼
10
OTel Emit + Log
Trace span (GenAI) + 8 metrics + structured JSON log
src/otel/{setup,llm_spans}.py → Tempo + Prometheus + Loki
