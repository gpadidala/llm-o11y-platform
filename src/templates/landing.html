Connect your providers

Add API keys for any LLM provider. Create virtual keys with per-team budgets, rate limits, and model-level permissions.

  • ✓ 6 providers: OpenAI, Anthropic, Vertex AI, Bedrock, Cohere, Azure
  • ✓ Virtual keys with sk-llmo-xxx format
  • ✓ Per-key budgets and spend tracking
  • ✓ Provider and model-level permissions
Create a virtual key
POST /api/keys { "name": "prod-key", "owner": "ml-team", "budget_usd": 100.00, "allowed_providers": ["openai"] } → { "key": "sk-llmo-a1b2c3..." }

Route intelligently

Six routing strategies automatically pick the best model based on cost, latency, or reliability. Built-in fallback chains and canary testing.

  • ✓ Cost-optimized: route to cheapest model
  • ✓ Latency-optimized: route to fastest provider
  • ✓ Fallback chains: auto-retry on failure
  • ✓ Canary: A/B test new models with traffic splits

Guard every request

18 PII detection patterns catch emails, phone numbers, SSNs, credit cards, and API keys before they reach the LLM. Content safety and output validation built in.

  • ✓ 18 PII patterns with automatic redaction
  • ✓ Content safety filtering
  • ✓ Topic restriction with word-boundary matching
  • ✓ JSON schema and regex output validation
PII Redaction
POST /api/guardrails/redact { "text": "Email john@acme.com, SSN 123-45-6789" } → "Email [EMAIL_REDACTED], SSN [SSN_REDACTED]"
